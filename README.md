# Study Flow

Study Flow é uma plataforma desenvolvida para auxiliar estudantes na organização de suas atividades acadêmicas e rotina de estudos. O sistema foi criado com foco em produtividade, praticidade e acessibilidade, permitindo que os usuários gerenciem suas informações de forma simples e eficiente.

## Objetivo

O projeto tem como objetivo ajudar estudantes a planejarem seus estudos, organizarem tarefas e acompanharem suas atividades acadêmicas em um único ambiente digital.

## Funcionalidades

- Organização de tarefas e atividades;
- Planejamento da rotina de estudos;
- Armazenamento de dados em nuvem;
- Interface responsiva para diferentes dispositivos;
- Navegação intuitiva e fácil de utilizar;
- Gerenciamento de informações acadêmicas.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Firebase
- Vercel
